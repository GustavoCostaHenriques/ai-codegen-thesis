# n8n Workflow Progress

This directory stores n8n workflow JSON exports used to track orchestration progress over time.

## Directory Layout

```text
n8n-progress/
`- daily/  # Daily workflow JSON snapshots
```

## File Naming Convention

Pattern:

- `YYYY-MM-DD_workflow-name_vN.json`

Examples:

- `2026-03-06_weekly-planning-sync_v1.json`
- `2026-03-07_weekly-planning-sync_v2.json`
